# Emerald Clean Pros

A professional cleaning service website for Seattle, Washington.

## About

Emerald Clean Pros is a locally-owned cleaning business serving the greater Seattle area since 2009. We offer house cleaning, deep cleaning, office cleaning, move-in/out cleaning, and eco-friendly cleaning options.

## Features

- **Responsive Design** - Works on all devices
- **Modern Pacific Northwest Aesthetic** - Emerald green, misty gray, clean white theme
- **Service Showcase** - Detailed service cards with features
- **Transparent Pricing** - Clear pricing tiers
- **Booking Form** - Easy quote request system
- **Smooth Animations** - Scroll animations and interactions

## Live Site

🌐 [https://emerald-clean-seattle.pages.dev](https://emerald-clean-seattle.pages.dev)

- HTML5
